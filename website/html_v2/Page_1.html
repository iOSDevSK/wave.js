Architecture of fluid.

Designed from the ground up for modern JS runtimes. Wave.js bypasses standard DOM manipulation to write directly to the GPU via custom GLSL shaders.

Steady 60 FPS

Highly optimized draw calls. Calculations are batched and executed natively on the GPU, leaving your main thread entirely free for application logic.

4.2kb

Micro Footprint

Zero dependencies. Tree-shakeable exports mean you only bundle what you render.

Agnostic Wrappers

First-class React, Vue, and Svelte components included. Or use vanilla JS.

Custom GLSL Inject

Override default vertex and fragment shaders. Pass custom uniforms directly through the JS config object.
