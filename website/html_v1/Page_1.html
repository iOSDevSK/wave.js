Simple to use.
Endlessly customizable.

Initialize the shader on any DOM element. Framework agnostic design.

// 1. Import the library
import Wave from 'wavejs';

// 2. Initialize on a container
const myWave = new Wave({
  el: '#hero-canvas',
  preset: 'liquidChrome',
  color1: '#00f0ff',
  color2: '#4d7cff',
  amplitude: 45.0
});

// 3. Profit globally 🌊

Zero Dependencies

Written in pure WebGL and vanilla JS. Under 8kb minified + gzipped. No bloat, just waves.

GPU Accelerated

Offloads rendering entirely to the GPU via custom fragment shaders. Rock solid 60fps on mobile.

Framework Agnostic

Works perfectly with React, Vue, Svelte, Angular, or just a plain HTML file. Wrappers included.

20+ Parameters

Total control over amplitude, frequency, zoom, phase, color stops, noise seeds, and blending modes.

TypeScript First

Built from the ground up in TS. Comprehensive type definitions and JSDoc comments built-in.

MIT Licensed

Truly open source. Free to use forever in personal, commercial, or client projects with no attribution required.
